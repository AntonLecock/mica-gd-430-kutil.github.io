Homework

  1. Create a Sitemap for Personal Website
    Needs to include sections for:
    • Homepage
    • Favorite Internet Things - list and detail view of things you find on the internet
    • Portfolio - list and detail view
    Outside of that, you can add anything additional that you want.
  2. Design the navigation for your personal website
    Using the sitemap you just created, design that primary navigation. Refer back to the presentation on navigation for different types.
  3. Favorite Internet Thing
    Find a website that uses hyperlinks in an interesting way. It can be in the way the site is structured (sitemap/navigation), their design, style (css), or interaction (js). With that website, do the following:
    1. Create a new HTML page on your website
    2. Give it a title
    3. Add the name of the website, and link to the website
    4. Insert a screenshot of the website
    5. Write 3-6 sentences about why you were drawn to this site
